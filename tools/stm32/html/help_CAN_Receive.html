S-Function

Nb Input: 0

Nb Output: 0 for interrupt receive mode and 5 for pooling receive mode.

  • Std/Ext for Standard (0) or Extended (1) data value
  • IDE Standard (11-bit) and Extended (29-bit)
  • D/R Data frame(0)/Request frame(2)
  • NbD Number of available data (0 to 8)
  • Data 8 char array. NbB significative data.

CAN receive parameters

top

CAN Name

Name of CAN communication channel selection. (CAN or CAN1/CAN2 Depending on STM32CubeMx configuration)

FIFO

FIFO communication selection : CAN_FIFO0 or CAN_FIFO1

Interrupt mode

Communication mode information. Depends on STM32CubeMx configuration. Selected for Interrupt communication mode.

TimeOut

Time out value for pooling communication mode. Not configurable for interrupt communication mode. It is the max number of solver loop after while pooling communication without answer is in default.

Type

STM32 can be Master or Slave depending on STM32CubeMx configuration.

Text

Information about mode and how to get data.
