Nb Input: 0 or 1 Fifo Number (0/1)

Name of CAN communication channel selection. (CAN or CAN1/CAN2 Depending on STM32CubeMx configuration)

Input FIFO num

FIFO number (0/1) is an input port.

Communication mode information. Depends on STM32CubeMx configuration.
